Welcome to GoCalc

This is a fun math and backend project built in Golang, created for portfolio and resume purposes.

It features a collection of mathematical tools with a focus on clean API design and frontend integration. The backend is written in Go, using performant techniques and concurrency where appropriate.

Available Tools:

  • 🧮 Calculus: limits, derivatives, integrals
  • 📐 Geometry: area and perimeter of shapes
  • 🧭 Coordinate Conversion: Cartesian, Polar, and Spherical

Feel free to explore each section using the navigation bar above. This project showcases backend service design, API handling, mathematical processing, and frontend rendering—all built from scratch.
